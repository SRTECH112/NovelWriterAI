### 3. Create API Key

2. Click "Create API Key"
3. Give it a name (e.g., "Novelist AI")

### 4. Add to Your .env.local File
Open `.env.local` and paste your key:

```env
AI_PROVIDER=GROQ

```

### 5. Restart the Dev Server
The server should auto-reload, or restart it manually:
